How to Clone a Website

- simple guide -

Tools

  1. Cyotek WebCopy https://www.cyotek.com/cyotek-webcopy
  2. HTTrack http://www.httrack.com/
  3. Scrap Book ext for FireFox
  4. wget/curl

[1] wget --no-check-certificate --recursive --page-requisites --convert-links --html-extension
[2] wget -r -l 3 -k -p -t 3 -nc

Example 1

We want to clone this site https://bunnycdn.com/

STR

1. Install Cyotek WebCopy -> (pic1)

2. Open WebCopy, paste site URL and select directory, which you want save site (pic2), pressed the button Copy and wait a minute

STR

3. Then pressed Open Output Folder (pic3) and convert all files with extension .htm to .html or remove .htm from filename: index.html -> index (pic4)

4. Inside all html files remove GTM/GA/ no needed scripts/css and WebCopier info (pic5)

STR

5. Use regexp, like: href=".*?" and remove no needed attributes or replace all tag href value (pic6) 

<a href="some_url">LINK</a>

6. Insert your GTM/GA and LINK you needed.
